# JS Drums
Play a drum set in your web browser!

### How to use
* [Download the zip](https://github.com/sean-capper/JS-Drums/archive/master.zip)
* Extract the files to a folder
* Open index.html in your browser of choice
* Use the keys displayed on the page to make music!
